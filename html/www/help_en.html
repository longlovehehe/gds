Account - Added user
Current user Current user = Total user - Lost user
Lost user The No. of users that not login in 14 consecutive days(exclude new created user in 14 days) + Deleted user in the query date
Total user In query date, the Activated users + Deleted users
proportion (Current user/Total user) * 100%
Account - Retention
Added user Created user per day; (create and delete in the same day won't be counted)
Deleted user Deleted user per day; (create and delete in the same day won't be counted)
Net growth Added user - Deleted user; Could be negative number

Activity
Activity - Online
Online When activity user login, then it will be marked as [Online], once per day; If the user has logged and without logout during a day, it will be seemed as [Online] too.
Offline It will be remark as [Offline] if activity user is offline and without login during the day
Total Total = Online + Offline
Activity - Continuity online
Continuity online N days A user with [Online] N days continuously
Activity - Activity
Active user A user is [Onlin] in a day, then it will be activive user too
Total user Total user = Activity user + Deleted user [in each day]
Activity (Active user/Total user) * 100%
